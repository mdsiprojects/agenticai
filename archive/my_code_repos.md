## 🔍 How to Use

For each repo, you can:
1. Clone it ⬇️
2. Create a virtual environment and install dependencies in `requirements.txt` 🌍
3. Follow the README 📖
4. Update your `.env` file with your credentials 🔑
5. Play with the code 👨‍💻
6. Add your own improvements ✨
